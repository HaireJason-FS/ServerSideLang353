# Geospatial Data API Integration + MongoDB

## Objective
Node.js REST API that:
1) Fetches geospatial data from a public API (Open-Meteo)
2) Stores geospatial data in MongoDB
3) Provides endpoints to fetch live data and retrieve stored data later

## Tech
- Node.js (18+)
- Express
- MongoDB + Mongoose
- Public API: Open-Meteo (no API key required)
